# 📋 社工紀錄產生器原型 (Social Work Record Generator)

> 「專為社工夥伴設計的紀錄詞彙生成與複製工具，旨在透過結構化引導，簡化重複性的文書作業流程。本專案目前已完成 Vue 3 溫暖風格重構，此後停止開發，作為歷史記憶的封存版本。」
> 
> **狀態：已封存 / 概念驗證 (Archived / Stable Prototype)**

---

## 🎨 介面與功能特色

1. **全新 Vue 3 (CDN) 輕量架構**：
   * 本產生器採用 CDN 方式載入 Vue 3，**不需要經過複雜的打包與建置（Vite Build）**。它是一個完全獨立、雙擊檔案即可在任何瀏覽器直接執行的輕量網頁，具備極佳的跨平台與移植便利性。
2. **溫暖配色與晶片按鈕 (Chip Elements)**：
   * 捨棄了傳統生硬的下拉選單，改為現代化、溫柔且極具質感的手風琴（Collapse）折疊面板，搭配溫暖色調的晶片按鈕。點選晶片即可即時將詞彙加入生成區域。
3. **擴充至 350 個專業詞彙（中英雙語對照）**：
   * 配合社工實務、法律背景及使用者的跨領域特質，收錄共 7 大分類，並且將每個分類都**精準擴充至 50 個簡短且實用兼具的中英對照詞彙**：
     * 🏹 反曲弓射箭 (Modern Recurve Archery)
     * ⚖️ 法律學碩士生 (LL.M. Student)
     * 🤝 社會政策與社會工作 (Social Policy & Social Work)
     * 🤖 工業自動化控制 (Industrial Automation Control)
     * ⚡ 電機工程 (Electrical Engineering)
     * 💻 資訊專員日常 (Information Specialist Daily)
     * 💬 紓壓與趣事 (Stress Relief & Fun Facts)
4. **一鍵複製與 Toast 提示**：
   * 產生的紀錄文字提供一鍵複製到剪貼簿功能，並配有防重複觸發、優雅漸變的 Toast 提示訊息，大幅優化文書產出的流暢度。
